Hey there! Thanks for checking this little project out. Let me give you a quick tour. This site is intended to run from a local docker process, but if you're reading this, you probably already figured that out.

When I work on little sample projects like this, I like to try to incorporate a few things at a time that I am interested in learning. In this case, I primarily wanted to explore Datastar, a new hypermedia library. If you don't know what any of that means, don't worry, I'll explain in more detail on the Datastar detail page.

The main thing I wanted to play with in Datastar is its use of Server Sent Events (SSE) to push updates to the client. Having created event-sourced systems in the past, getting the client to become "eventually consistent" with new events which have been published since the page was loaded has always been a challenge.

Having been developing software primarily in JavaScript, I also wanted to build a backend with Python. I'll go into some detail around how that backend is set up and how the events are sent to the browser.

Finally, I wanted to use the US Web Design System (USWDS) to build the UI.
